# Agro Sustentável 🌱

Jogo desenvolvido em p5.js com foco no equilíbrio entre produção agrícola e sustentabilidade ambiental.

## Objetivo

Colete:

- 🌱 Árvores → aumenta sustentabilidade
- 💧 Água → aumenta produção
- 🛢️ Poluição → reduz sustentabilidade

Ao final de 60 segundos, o resultado será calculado de acordo com seu desempenho.

## Como jogar

- ⬅️ Seta esquerda: mover para a esquerda
- ➡️ Seta direita: mover para a direita

## Tecnologias

- HTML5
- CSS3
- JavaScript
- p5.js
